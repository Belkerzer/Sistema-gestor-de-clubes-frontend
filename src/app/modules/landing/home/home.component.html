Módulo de aterrizaje

Esta puede ser la página de aterrizaje o de bienvenida de su aplicación. Si tienes una configuración de SSR (Server Side Rendering), o si no necesitas tener visibilidad en los motores de búsqueda y optimizaciones, puedes incluso usar esta página como tu página de aterrizaje principal.

Este es un "módulo" separado, tiene su propio directorio y configuración de enrutamiento y también está completamente separado de la aplicación real. Este es también un ejemplo simple de una configuración de múltiples aplicaciones que utiliza el mismo código base. Usted puede tener diferentes puntos de entrada y esencialmente tener diferentes aplicaciones dentro de la misma base de código.
